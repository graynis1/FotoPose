# App Icon

Drop a single `AppIcon-1024.png` (1024×1024, no alpha, no transparency) into
this folder. Xcode 14+ automatically generates every other size from it.

Brand guidance:
- Gradient background: `#A78BFA` → `#EC4899` (violet to pink, diagonal).
- Centered glyph: white `sparkles` SF Symbol or a hand-drawn "P" monogram.
- 180×180 @2x / 1024×1024 base; corners left unrounded (iOS masks them).
